# 실행 방법
```
npm install
npm run dev
```
개발 서버는 http://localhost:3000에서 실행됩니다.

# 프로젝트 구조
```
src/
├── app/                 # Next.js 앱 라우터
    ├── components/      # 재사용 가능한 UI 컴포넌트
    └── transactions/    # 전체 거래 내역 페이지
└── lib/                 # 공통 라이브러리
    ├── api/             # API 호출 및 데이터 fetching
    ├── chart/           # Chart.js 설정 및 옵션
    ├── hook/            # 커스텀 React Hooks
    ├── store/           # Zustand 전역 상태 관리
    ├── types/           # TypeScript 타입 정의
    └── utils/           # 유틸리티 함수
```
# 디자인 의도 및 UI/UX 포인트

별도의 대시보드 템플릿을 사용하지 않고 직접 디자인을 구성했습니다.

- 정보 계층 구조: 가장 중요한 요약 지표를 상단 카드로 배치하고, 시각적 데이터(차트), 상세 데이터(테이블) 순으로 정보의 우선순위를 시각적으로 구분했습니다.
- 가독성 중심: Tailwind CSS를 활용해 충분한 여백과 명확한 색상 구분으로 데이터를 쉽게 파악할 수 있도록 설계했습니다.
- 실용적 인터랙션: 필터 검색과 페이지네이션을 통해 대량의 거래 데이터를 효율적으로 탐색할 수 있도록 구성했습니다.


## 주요 기능
### 1. 메인 대시보드
```
- 요약 카드 4개
   1. 오늘 결제 금액
   2. 거래 건수
   3. 결제 완료율(%)
   4. 결제 실패 & 환불 건수


- 차트 2개
   1. 일별 거래 금액 추이 (최근 7일)
   2. 금일 결제 상태 분포


- 최근 거래 목록 (5건)
   1. 결제일시
   2. 가맹점
   3. 결제수단
   4. 상태
```

### 2. 전체 거래 내역
```
- 필터 검색 기능
  1. 날짜, 가맹점, 결제수단, 상태 등으로 검색


- 전체 거래 내역 테이블
  1. 페이지네이션 지원
  2. 거래 상세 정보 표시
  3. 가맹점 클릭 시 팝업으로 가맹점 상세 정보 확인 가능
```
